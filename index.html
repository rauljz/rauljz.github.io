K-Means Clustering

Unsupervised grouping algorithm

+

Concept Overview

K-Means is an unsupervised Machine Learning algorithm used to partition unlabeled data into K distinct clusters based on feature similarity (using Euclidean distance).

Key Highlights

  • Iterative refinement of cluster centroids.
  • Minimizes within-cluster variance (inertia).
  • Use cases: Customer segmentation, anomaly detection, and image compression.

Text Tokenizer

NLP text-to-vector preprocessing

+

Concept Overview

Large Language Models (LLMs) do not read text character by character. They use tokenizers to break text into manageable sub-words or chunks, mapping each to a unique token ID in their vocabulary space.

Key Highlights

  • Simulates sub-word tokenization (like BPE or WordPiece).
  • Transforms human text into numeric tensors for AI.
  • Relevance: Crucial for Prompt Engineering and LLMOps inference pipelines.

Object Detection

Computer Vision inference simulation

+

Concept Overview

Simulates the output of Convolutional Neural Networks (CNNs) like YOLO or Faster R-CNN. The model identifies regions of interest, applies bounding boxes, and assigns a classification probability.

Key Highlights

  • Spatial feature extraction and anchor boxes.
  • Confidence thresholds for predictions.
  • Use cases: Medical imaging (e.g., polyp detection), autonomous driving.

Gradient Descent

Mathematical optimization for Loss functions

+

Concept Overview

The core optimization algorithm behind Deep Learning. It updates model weights iteratively by moving in the direction of the steepest descent (negative gradient) of the Loss function landscape.

Key Highlights

  • Visualizes a learning rate applied to a derivative.
  • Strives to find the global minimum.
  • Relevance: Backpropagation in Neural Networks.

Activation Functions

Introducing non-linearity in Neural Nets

+

Concept Overview

Without activation functions, a deep neural network would simply be a linear regression model. These functions decide whether a neuron should fire, enabling the network to learn complex patterns.

Key Highlights

  • ReLU: Fast and helps prevent vanishing gradients.
  • Sigmoid/Tanh: Great for probabilities but prone to saturation.

RAG Architecture

Retrieval-Augmented Generation Simulator

+

Concept Overview

RAG enhances Large Language Models by connecting them to external knowledge bases. Before answering, the system queries a vector database (like Pinecone or FAISS) to retrieve factual context.

Key Highlights

  • Reduces hallucinations by grounding answers in data.
  • Combines semantic search (embeddings) with generation.
  • Relevance: Enterprise AI systems and private-data chatbots.

CNN Convolutions

Mathematical Image Filtering

+

Concept Overview

Convolutional Neural Networks (CNNs) process images by sliding mathematical matrices called kernels (or filters) over pixels to extract spatial features like edges, corners, and textures.

Key Highlights

  • Reduces spatial dimensions while capturing deep features.
  • Core operation behind models like ResNet and YOLO.
  • Relevance: Polyp detection (medical imaging) and autonomous driving.
